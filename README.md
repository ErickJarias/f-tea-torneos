# Fútea Pro - Gestión de Torneos de Élite ⚽

Fútea Pro es una plataforma integral para la organización y gestión de torneos de fútbol, diseñada para ofrecer una experiencia profesional tanto a organizadores como a participantes.

## 🚀 Características Principales

- **Gestión de Torneos**: Creación y administración de torneos (Eliminatoria, Liga, etc.).
- **Panel de Control Pro**: Rediseñado para ser 100% responsive y fácil de usar en móviles.
- **Gestión de Sponsors**: Sube logos de patrocinadores directamente y gestiónalos por categorías (Oro, Plata, Bronce).
- **Dashboard de Usuarios**: Monitoreo de actividad de organizadores con alertas de inactividad.
- **Autenticación Segura**: Integración con Firebase Google Auth.
- **Base de Datos en Tiempo Real**: Sincronización instantánea con Firestore.

## 🛠️ Tecnologías Utilizadas

- **Frontend**: React 18 + Vite
- **Estilos**: Tailwind CSS + shadcn/ui
- **Backend/Base de Datos**: Firebase (Firestore & Auth)
- **Iconografía**: Lucide React
- **Animaciones**: Framer Motion

## 📦 Estructura del Proyecto

El proyecto sigue una estructura moderna de React:
- `src/components`: Componentes de la interfaz (UI).
- `src/lib`: Utilidades y configuración.
- `src/firebase.ts`: Configuración de la conexión con Firebase.
- `src/App.tsx`: Lógica principal de la aplicación.

## 🔧 Configuración Local

1. Clona el repositorio.
2. Instala las dependencias: `npm install`.
3. Configura tus credenciales de Firebase en `src/firebase-applet-config.json`.
4. Inicia el servidor de desarrollo: `npm run dev`.
